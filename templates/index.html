Control Panel

for StartApp

If you want to change the settings, edit into the config file.

  • Clone: Clone's the git URL that you have set into the directory you have set. (If already cloned it will update the updated files in the repo to hopefully prevent the risk of dataloss)
  • Execute: Run the executable at the directory set with the command set.
  • Quit: Forcefully stops the executable.
  • Restart: Restarts the executable.
  • Quit StartApp: Quits StartApp. (Also quits the executable)
